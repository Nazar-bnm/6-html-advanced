1) Атрибути елементів форм:

autocomplete - браузер запам'ятовує, введену раніше інформацію. 
				При вводі перших букв пропонуються варіанти з яких можна вибрати потрібний.
				Цю функцію можна відключити в настройках браузера.
				Має два значення on - включити і off - виключити автозапам'ятовування.
				Можна задати для form autocomplete="on", а для поля autocomplete="off", чи на оборот.


multiple - дозволяє вибрати одночасно декілька файлів для загрузки на сайт, наприклад фотографій.
			А також для введення електронної пошти, які мають перераховуватися через кому.
			Якщо атрибут multiple доданий до тегу <select>, то показує не один пункт для вибору а декілька (в хромі 4) з прокруткою.
			Кількість пунктів можна вказати атрибутом size="3"

placeholder - виводить слова підказки на фоні поля для вводу, які зникають при початку набору тексту.

pattern - задає шаблон для поля вводу. Форма не буде відправлена поки дані не відповідатимуть шаблону.

autofocus - ставить курсор в потрібне поле при загрузці сторінки.
			Якщо це кнопка, то на неї можна нажати відразу за допомогою клавіатури(Enter), без наведення мишкою.

readonly - елемент форми призначений, тільки для читання і не може бути змінений. 
			Просто пишеться readonly, без значень.

disabled - відключає елемент форми, фон поля для вводу стає сірим і неактивним.
			Просто пишеться disabled, без значень.

required - показує що поле обов'язкове для заповнення.

2) Типи елементів форм:

range - повзунок, для вибору числового значення з певного діапазону.
tel - призначений для вводу телефонних номерів.
time - призначений для вводу часу.
url - призначений для вводу веб адрес.
number - для вводу чисел. Можна вказати обмеження min="3" max="21", а токож крок step="3"
radio - для вибору одного варіанту з декількох.
checkbox - для можливості вибрати декілька варіантів.
submit - кнопка для відправки даних на сервер.
button - кнопка.
date - для вводу дати. Також можна ставити обмеження, наприклад min="1950-01-01" max="2000-12-31"
text - для вводу тексту.
password - для паролів.
color - для вибору кольору.
search - для пошуку.
reset - для скидання даних.
file - для відправки файлів.

-----------------------------------------------------------------------------------------------------------------

3) Опис тегів:

dl - контейнер для списку в форматі термін-значення. Працює з тегами dt і dd.
dt - термін, який описується.
dd - визначення терміну.

fieldset - створює рамку для групування елементів форми по контексту. 

legend - створює заголовок, який виводиться в розриві рамки.

datalist - створює список варіантів для вибору в формі. Привводі тексту появляються варіанти.
			"прив'язується" до input через id.
			Варіанти списку записуються в <option value="item">

optgroup - контейнер для <option>. Створює список варіантів для вибору об'єднаних в одну групу.
			Розміщується в середині <select>



progress - використовується для відображення прогресу виконання задачі. 
			Зміна значень відбувається через JavaScript.


 meter - для виводу числового значення в певному діапазоні у вигляді полоски, яка змінює розмір і колір 
 			в залежності від значення.

iframe - створює рамку, яка знаходиться в середині звичайного документа і дозволяє загружати в цю область
		 любі інші незалежні документи.

embed - використовується для загрузки і відображення об'єктів (відофайлів, флеш роликів)

object - використовується для вставки об'єктів, такі як зображення, аудіо, відео, Java-апплетів, 
		ActiveX, PDF и Flash.
